Por onde começar com PHP

O PHP é uma das linguagens de programação mais utilizadas no desenvolvimento web por ser multiplataforma, open source, gratuita e bastante completa. A linguagem PHP conquistou o mercado e os seus desenvolvedores, tendo uma comunidade vibrante e uma evolução rápida nos últimos anos.

A Formação PHP da Alura é um guia de aprendizado completo para você iniciar uma carreira bem sucedida como programador PHP. Você vai aprender os fundamentos da linguagem, as boas práticas, a orientação a objetos e os frameworks populares, como Doctrine, utilizado para acessar um banco de dados de maneira produtiva.

Você também irá utilizar Laravel/Lumen e Symfony para a construção de aplicações web e APIs, o Composer para controlar as dependências do projeto e muito mais!
